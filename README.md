# Backtester
Performed several statistical tests like adfuller,kpss to check the stationarity of the AAPL stocks.Performed ARIMA on AAPL stocks and also
performed ljung-box test to and also predicted future values using kalman filtering and Holt - Winter Smoothing.
Performed Backtesting on AAPL stocks by using appropriate trading strategies to get 27 percent on 10 years of AAPL stocks . Also computed
several indicators like ATR,SMA,MACD,RSI etc.
